# 📊 Telecom Network Customer Analytics

## 📌 Project Overview
This project analyzes telecom customer data using Excel, SQL, Python, and Tableau to generate business insights related to customers, network usage, and service issues.

---

## 🎯 Objectives
- Analyze telecom customer data.
- Perform SQL-based data analysis.
- Clean and process data using Python.
- Build an interactive Tableau dashboard.
- Present key business insights.

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- SQL
- Python
- Tableau Public

## 📈 Tableau Dashboard

The dashboard includes:

- Total Customers KPI
- Total Issues Reported KPI
- Total Bills Generated KPI
- Customer Distribution by City
- Network Type Distribution (4G vs 5G)
- Issue Type Distribution

---

## 📊 Key Insights
- Customer distribution across major cities.
- Comparison of 4G and 5G network usage.
- Analysis of common network issues.
- Summary KPIs for customers, issues, and billing.
